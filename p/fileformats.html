Listing File Formats

There are many different file formats that are recognized by various operating systems. This is a (currently partial) list of them, organized roughly by popularity. Most of the data here was found on the Wikipedia page on this topic.

.7z7-Zip compression
.debDebian Linux package file
.dmgArchive type used by Apple
.exeA program for Windows
.gzG-Zipped file
.binAn executable
.pyPython file
.tarPackage of files
.xzA compression format used by LZMAL
